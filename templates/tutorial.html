Cách cài Việt Hóa & Font cho PC và Android chi tiết nhất

Quay lại trang chủ
Phần 1: Cài đặt Việt Hóa
1Tải File

Tải file ZIP Việt Hóa tương ứng với phiên bản game của bạn (Launcher/Steam/Epic) tại trang chủ.

2Giải nén

Sử dụng WinRAR hoặc 7-Zip để giải nén file vừa tải về. Bạn sẽ nhận được thư mục wwm_standard (hoặc lite).

3Chép vào thư mục game

Copy toàn bộ thư mục vừa giải nén vào nơi cài đặt game.

Lưu ý: Khi hệ thống hỏi có muốn Ghi đè (Overwrite/Replace) không, hãy chọn Yes/All.
Phần 2: Cài đặt Font chữ (Tùy chọn)

Để font chữ đẹp hơn và không bị lỗi ô vuông, bạn nên sử dụng công cụ Font Patcher của chúng tôi.

1Chuẩn bị file gốc

Vào thư mục game, tìm file Resources.mpk (thường nằm trong thư mục gốc hoặc thư mục Data).

2Sử dụng Tool

Truy cập Công cụ chỉnh Font trên website này.

  • Upload file Resources.mpk.
  • Upload file Font .ttf mà bạn muốn dùng (Arial, Times New Roman, v.v...).
  • Nhấn nút Tạo File.
3Cài đặt

Sau khi tải file WWM_VietHoa_Full.zip về, hãy giải nén và chép đè vào thư mục game tương tự như cài Việt Hóa.

Hướng dẫn này dành cho Android 11 trở lên (đặc biệt là Android 16) - nơi việc truy cập thư mục Data bị hạn chế.

Các bước thực hiện chi tiết

1Cấu hình Shizuku (Quan trọng)

Đây là bước để "bẻ khóa" quyền truy cập thư mục hệ thống.

  1. Mở Shizuku > Chọn mục Ghép nối (Pairing).
  2. Vào Cài đặt điện thoại > Tùy chọn nhà phát triển (Developer Options).
  3. Tìm và bật Gỡ lỗi không dây (Wireless Debugging).
  4. Bấm vào chữ "Gỡ lỗi không dây" > Chọn "Kết nối thiết bị bằng mã ghép nối".
  5. Ghi nhớ mã số 6 chữ số hiện ra. Kéo thanh thông báo xuống, chọn thông báo của Shizuku và nhập mã đó vào.
  6. Sau khi báo thành công, quay lại Tùy chọn nhà phát triển, bật thêm:
    • Gỡ lỗi USB.
    • Cài đặt bảo mật (Security Settings) (nếu có).
    • Tắt giám sát quyền (nếu có).
  7. Quay lại App Shizuku, bấm Bắt đầu (Start). Nếu hiện "Shizuku đang chạy" là OK.
2Cấp quyền cho ZArchiver
  1. Mở ZArchiver.
  2. Vào Cài đặt (Dấu 3 chấm góc phải) > ROOT.
  3. Tích chọn vào các ô tùy chọn liên quan đến Shizuku/Root trong đó.
  4. Lúc này ZArchiver sẽ hỏi quyền truy cập thông qua Shizuku > Chọn Cho phép mọi lúc.
3Tải & Cài đặt Việt Hóa
  1. Mở game Where Winds Meet Mobile ít nhất 1 lần để game tải khoảng 1GB dữ liệu đầu (tạo thư mục Data).
  2. Tải file Việt Hóa về máy.
  3. Mở ZArchiver, tìm đến file vừa tải và giải nén.
  4. Tìm file có tên: translate_words_map_...
  5. Nhấn giữ vào file đó > Chọn Sao chép (Copy).
  6. Di chuyển đến thư mục Data của game (thường là: Android/data/com.netease.z55.../files/...).
  7. Dán đè (Paste) file vào. Chọn Thay thế (Overwrite) nếu được hỏi.
